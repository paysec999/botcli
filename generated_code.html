About Me

This is a simple example of an HTML webpage. You can use HTML to create websites with text, images, links, and more. This example includes a header, navigation bar, main content area, and a footer. The styling is done using internal CSS within the <style> tags in the <head> section.

You can easily customize this template to create your own unique website. Just edit the HTML and CSS code to your liking.

Key HTML Elements:

  • <!DOCTYPE html>: Declares the document type as HTML5.
  • <html>: The root element of the HTML page.
  • <head>: Contains meta-information about the HTML page.
  • <body>: Contains the visible page content.
  • <header>: Specifies a header for a document or section.
  • <nav>: Defines a set of navigation links.
  • <main>: Specifies the main content of a document.
  • <footer>: Specifies a footer for a document or section.
  • <h1> - <h6>: HTML headings.
  • <p>: HTML paragraph.
  • <a>: HTML link (hyperlink).
  • <ul>, <ol>, <li>: HTML lists.
